# estrutura_dados2

Este repositório contém a implementação de estruturas de dados avançadas e algoritmos de manipulação de arquivos, 
focando em eficiência de memória e tempo de execução.

### 🛠️ Tecnologias Utilizadas
- Linguagem: C++17 ou superior.
- Compilador: GCC/G++.
- Bibliotecas Standard: <iostream>, <fstream>, <queue>, <stack>, <vector>.
